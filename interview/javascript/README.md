# JavaScript 面试基础练习

## 如何使用

1. 先看 `01_var_let_const.js`，理解作用域和提升。
2. 再看 `02_promise_async.js`，理解 Promise 链和 async/await。
3. 最后做 `03_fundamentals_katas.js`，按 TODO 手写基础题。

## 运行方式

```bash
node interview/javascript/01_var_let_const.js
node interview/javascript/02_promise_async.js
node interview/javascript/03_fundamentals_katas.js
```

## 练习建议

- 每个文件先自己口述一遍“概念 + 常见坑 + 示例”。
- 做完 TODO 后，自己补一组反例测试（空输入、null、重复值）。
- 面试时优先说“为什么这样设计”，再说代码细节。
